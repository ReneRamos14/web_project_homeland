# TripleTen Homeland

Proyecto de práctica de HTML y CSS para mostrar ciudades natales de empleados.

## Vista del proyecto

Puedes ver el proyecto en GitHub Pages:

[https://reneramos14.github.io/web_project_homeland/](https://reneramos14.github.io/web_project_homeland/)

## Estructura del proyecto

- `index.html` → página principal  
- `pages/` → CSS principal  
- `blocks/` → CSS por bloques  
- `images/` → imágenes utilizadas  
- `vendor/` → fuentes y normalize.css
